## Troubleshooting

**"I don't see 'Manage messaging & content on Instagram'"**
- Make sure you're looking at the full list (select "All (19)")
- Try filtering by "Content management"
- Scroll down - it might be further in the list
- Refresh the page and try again

**"The checkbox is grayed out"**
- Make sure you're logged into Meta for Developers
- Try a different browser
- Clear cache and cookies

**"I selected the wrong use case"**
- You can delete the app and create a new one
- Or contact Meta support to change the use case
